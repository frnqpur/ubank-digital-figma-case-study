# Video Demo Checklist — UBank Digital

## Goal
Video demo digunakan untuk memberi recruiter gambaran cepat tentang prototype tanpa harus langsung membuka Figma.

Durasi ideal: **60–90 detik**.

## Existing Asset
Aset video yang tersedia:

```text
Prototype Aplikasi UBank Digital.mp4
```

Video asli dapat dipakai sebagai sumber, tetapi sebaiknya dibuat versi pendek untuk portfolio.

## Recommended Video Structure
### 0–5 detik — Opening
Tampilkan:
- Project title: UBank Digital
- Label: Mobile Banking UI/UX Prototype
- Figma prototype preview

### 5–20 detik — Onboarding
Tampilkan:
- Splash screen
- Onboarding screen 1
- Onboarding screen 2
- Onboarding screen 3

Narasi singkat:
> The onboarding introduces the key value proposition of a simple and secure mobile banking experience.

### 20–35 detik — Authentication
Tampilkan:
- Sign in
- Register

Narasi:
> Users can enter the product through familiar sign-in and registration screens.

### 35–55 detik — Dashboard & Cards
Tampilkan:
- Home dashboard
- My cards
- All cards
- Statistics

Narasi:
> The dashboard presents financial information through card-based UI, quick actions, and transaction summaries.

### 55–75 detik — Transactions & Send Money
Tampilkan:
- Transaction history
- Search transaction
- Send money
- Payment detail

Narasi:
> The transaction and send money flows help users review activity and complete common banking tasks.

### 75–90 detik — Settings/Profile
Tampilkan:
- Profile
- Settings
- Language / change password jika perlu

Narasi:
> Profile and settings screens support account management and user preferences.

## Required Video Quality
- [ ] 1080p export if possible
- [ ] Smooth screen transitions
- [ ] No visible private data
- [ ] No long idle moments
- [ ] No unrelated desktop clutter
- [ ] Clear cursor movement if recording from Figma
- [ ] Audio optional, captions preferred
- [ ] File size optimized for portfolio website
- [ ] Filename clean and lowercase

## Recommended Filename
```text
ubank-digital-figma-prototype-walkthrough.mp4
```

## Optional Caption Text
```text
UBank Digital — Mobile Banking UI/UX Prototype
Designed in Figma
Onboarding • Auth • Dashboard • Cards • Transactions • Send Money • Settings
```

## Do Not Show
- Full card number
- CVV
- Real email
- Real phone number
- Personal identity data
- Unfinished or inconsistent screens
- Claims about real payment processing
- Claims about backend integration
