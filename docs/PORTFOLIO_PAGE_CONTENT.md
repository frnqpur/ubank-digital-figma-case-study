## Overview
UBank Digital is a mobile banking UI/UX prototype that explores how users can access key financial features through a clean and structured mobile experience. The design covers the main user journey from onboarding and authentication to dashboard, card management, transaction history, send money, and profile/settings.

This project was created as a design prototype in Figma and prepared as a portfolio-ready case study.

## Problem
Mobile banking users often need to complete financial tasks quickly, but banking interfaces can feel complex when they contain too much information at once. Users need clear navigation, visible financial summaries, and simple flows for common actions such as checking transactions and sending money.

## Goal
The goal was to design a mobile banking prototype that:

- Introduces the product clearly through onboarding
- Provides a simple sign-in and registration experience
- Displays key financial information on the dashboard
- Supports card management
- Helps users review and search transactions
- Provides a clear send money flow
- Includes profile and settings management

## User Flow
```text
Splash
→ Onboarding
→ Sign In / Register
→ Home Dashboard
→ Statistics
→ My Cards
→ Transaction History
→ Search Transaction
→ Send Money
→ Profile / Settings
```

## Key Screens
### Onboarding
Introduces the value proposition of fast, secure, and convenient mobile banking.

### Authentication
Provides sign in and registration screens using familiar mobile form patterns.

### Dashboard
Shows card information, quick actions, and recent financial activity in a card-based layout.

### Card Management
Allows users to view card information and manage their cards through dedicated screens.

### Transaction History
Displays financial activity in a structured list with merchant/category context.

### Search Transaction
Helps users find specific transaction records more efficiently.

### Send Money
Guides users through recipient selection, amount input, and payment detail review.

### Profile & Settings
Supports account management, user information, language settings, and password changes.

## Design System
The visual system uses:

- Primary blue for CTA and highlights
- Clean white background
- Card-based financial layout
- Bottom navigation
- Rounded cards and buttons
- Simple form inputs
- Transaction list pattern
- Financial dashboard hierarchy
- Icons and illustrations for context

Before publishing publicly, the project should be cleaned by fixing typos, masking dummy data, standardizing currency format, and exporting high-quality screenshots.

## Disclaimer
This project is a UI/UX design prototype created in Figma. It does not include backend, database, banking API integration, payment processing, or real banking operation.
