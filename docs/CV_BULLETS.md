# CV Bullets — UBank Digital

## Project Title Options
Use one of these:

```text
UBank Digital — Mobile Banking UI/UX Prototype
UBank Digital — Figma Mobile Banking Case Study
UBank Digital — Fintech App UI/UX Design
```

## Short CV Bullet
```text
Designed UBank Digital, a Figma-based mobile banking UI/UX prototype covering onboarding, authentication, dashboard, card management, transaction history, send money, and profile/settings flows.
```

## Medium CV Bullet
```text
Created a mobile banking UI/UX prototype in Figma with 22 screens, including onboarding, sign in/register, dashboard, card management, transaction history, search transaction, send money, and settings/profile flows.
```

## Impact-Oriented CV Bullet
```text
Designed an end-to-end mobile banking prototype in Figma, translating core financial tasks into a structured user flow across onboarding, authentication, dashboard, cards, transactions, send money, and account settings.
```

## Portfolio-Oriented CV Bullet
```text
Built a recruiter-ready UI/UX case study for a mobile banking concept, including Figma prototype, video walkthrough, key screen exports, design system summary, and bilingual portfolio documentation.
```

## Indonesian CV Bullet

```text
Mendesain prototype UI/UX mobile banking di Figma dengan 22 screen, mencakup onboarding, login/register, dashboard, manajemen kartu, riwayat transaksi, pencarian transaksi, kirim uang, serta profil/pengaturan.
```

```text
Role: UI/UX Designer / Product Designer

```

## LinkedIn Project Description
```text
UBank Digital is a Figma-based mobile banking UI/UX prototype designed to explore a simple and modern financial app experience. The prototype covers onboarding, authentication, dashboard, card management, transaction history, send money, and profile/settings flows. This project demonstrates mobile app flow planning, financial UI layout, card-based interface design, and prototype presentation for portfolio review.
```

## Avoid These Claims
Do not write:

```text
Built a real banking app
Developed a production mobile app
Integrated banking APIs
Implemented payment processing
Built secure authentication system
Connected app to database
Launched fintech product
```

Use these instead:

```text
Designed a mobile banking prototype
Created a Figma UI/UX case study
Prepared an interactive design prototype
Designed the user flow and key screens
Created a video walkthrough and portfolio documentation
```
