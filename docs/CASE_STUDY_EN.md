# Case Study — UBank Digital

## Overview
UBank Digital is a mobile banking UI/UX prototype designed to create a simple, modern, and easy-to-understand digital banking experience. Built in Figma, the prototype covers the main user journey from onboarding and authentication to dashboard, card management, transaction history, send money, and profile/settings.

The project focuses on helping users quickly understand the product, access key financial information, review cards and transactions, and complete a send money flow through a structured mobile interface.

## My Role
My role in this project included:

- Structuring the main app flow
- Designing key mobile banking screens
- Creating the interactive Figma prototype
- Preparing visual assets for portfolio presentation
- Writing the case study and project documentation

## Problem
Mobile banking users need an app that helps them complete basic financial tasks quickly and clearly. However, financial apps can easily become complex because they need to display balances, cards, transactions, account details, and settings.

The design challenge was to organize these features into a clean and understandable mobile experience.

## Goal
The main goals were to:

- Create an easy-to-follow mobile banking flow
- Introduce the product through a clear onboarding experience
- Design a dashboard that summarizes key financial information
- Create a card management experience
- Design transaction history and transaction search screens
- Build a clear send money flow
- Provide profile and settings screens for account management

## Target User
The target users are mobile banking users who want to:

- View balance and card information
- Review transaction history
- Send money
- Manage profile and account preferences
- Access key features quickly from the dashboard

## User Flow
Main flow:

```text
Splash
→ Onboarding
→ Sign In / Register
→ Home Dashboard
→ Statistics
→ My Cards
→ Transaction History
→ Search Transaction
→ Send Money
→ Profile / Settings
```

## Key Screens
### 1. Onboarding
The onboarding screens introduce the key value of the product: fast, secure, and simple mobile banking.

### 2. Authentication
The sign-in and registration screens use simple form patterns to help users enter the app quickly.

### 3. Home Dashboard
The dashboard acts as the main hub, showing card information, quick actions, and transaction summaries.

### 4. Card Management
The card screens allow users to view and manage their digital cards.

### 5. Transaction History
The transaction history screen helps users review their financial activity.

### 6. Transaction Search
The search transaction screen helps users find specific transactions more efficiently.

### 7. Send Money
The send money flow helps users choose a recipient, enter an amount, and review payment details.

### 8. Profile & Settings
Profile and settings screens support account management, language preferences, password changes, and user information.

## Design System Summary
The visual direction includes:

- Primary blue color for CTAs and highlights
- Clean white background
- Card-based layout for financial information
- Bottom navigation for quick access
- Simple form inputs for authentication and transactions
- Icons and illustrations to support context
- Clear text hierarchy for important financial data

## Reflection
This project demonstrates the ability to structure a mobile app flow, design financial dashboard screens, create card-based UI patterns, and prepare a recruiter-friendly portfolio presentation.

Recommended improvements before publishing:

- Fix visible typos
- Standardize language
- Standardize currency format
- Mask dummy data such as card number, CVV, email, and phone number
- Use a shorter video walkthrough
- Highlight the strongest screens in the portfolio hero section

## Disclaimer
UBank Digital is a Figma UI/UX prototype. It does not include backend, database, banking API integration, payment processing, or real banking operation.
