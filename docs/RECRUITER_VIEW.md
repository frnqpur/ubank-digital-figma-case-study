# Recruiter View — UBank Digital

## One-Line Summary
UBank Digital is a mobile banking UI/UX prototype created in Figma, covering onboarding, authentication, dashboard, card management, transaction history, send money, and profile/settings flows.

## What Recruiters Should Notice
Recruiter sebaiknya langsung melihat bahwa project ini menunjukkan kemampuan dalam:

- Designing end-to-end mobile app user flow
- Building a clean financial dashboard experience
- Structuring onboarding for a fintech/banking product
- Designing authentication and form-based screens
- Creating card management UI
- Presenting transaction history and search pattern
- Designing send money flow
- Preparing Figma prototype and video demo
- Packaging UI/UX work into a clear portfolio case study

## Best Demo Format
Urutan demo terbaik:

1. **Figma Prototype**  
   Primary demo untuk menunjukkan interaksi dan flow.

2. **Video Walkthrough**  
   Demo cepat 60–90 detik untuk recruiter yang tidak ingin membuka Figma.

3. **Portfolio Case Study**  
   Penjelasan problem, goal, flow, key screens, design decisions, dan reflection.

4. **Screenshot Gallery**  
   Visual proof untuk website portfolio dan CV.

## Recommended Recruiter CTA
Gunakan CTA berikut di portfolio:

- Open Figma Prototype
- Watch Video Demo
- View Case Study
- See Key Screens

## What To Avoid
Jangan menampilkan project ini sebagai:

- Fully developed mobile banking app
- Production-ready fintech product
- Backend-integrated banking system
- Real payment application
- Application with real users or real transaction data

## Recruiter-Friendly Project Description
UBank Digital is a Figma-based mobile banking UI/UX prototype that explores a simple, modern, and structured financial app experience. The prototype covers the main user journey from onboarding and authentication to dashboard, card management, transaction history, send money, and profile/settings screens.

The project demonstrates my ability to design mobile app flows, organize financial information visually, create consistent UI patterns, and prepare an interactive prototype that can be reviewed through Figma and video walkthrough.
