# Studi Kasus — UBank Digital

## Ringkasan
UBank Digital adalah prototype UI/UX aplikasi mobile banking yang dirancang untuk memberikan pengalaman perbankan digital yang sederhana, modern, dan mudah digunakan. Prototype ini dibuat di Figma dan mencakup flow utama seperti onboarding, sign in, register, dashboard, card management, transaction history, send money, serta settings/profile.

Project ini berfokus pada bagaimana pengguna dapat memahami produk dengan cepat, mengakses informasi keuangan utama, melihat kartu dan transaksi, serta melakukan alur pengiriman uang melalui tampilan mobile yang terstruktur.

## Peran Saya
Dalam project ini, saya berperan dalam:

- Menyusun struktur flow aplikasi
- Mendesain screen utama mobile banking
- Membuat prototype interaktif di Figma
- Menyiapkan aset visual untuk portfolio
- Menyusun dokumentasi case study dan presentasi project

## Problem
Pengguna mobile banking membutuhkan aplikasi yang dapat membantu mereka melakukan aktivitas finansial dasar dengan cepat dan mudah. Namun, aplikasi keuangan sering terasa kompleks karena banyaknya informasi seperti saldo, kartu, transaksi, profil, dan pengaturan akun.

Tantangan desain dalam project ini adalah menyusun tampilan yang tetap informatif tetapi tidak membingungkan pengguna.

## Goal
Tujuan desain:

- Membuat flow mobile banking yang mudah dipahami
- Menyediakan onboarding yang menjelaskan nilai utama aplikasi
- Mendesain dashboard yang menampilkan informasi penting secara ringkas
- Membuat UI untuk card management
- Menyusun tampilan transaction history dan search
- Mendesain alur send money yang jelas
- Menyediakan profile dan settings untuk pengelolaan akun

## Target User
Target user adalah pengguna mobile yang ingin:

- Melihat saldo dan informasi kartu
- Mengecek riwayat transaksi
- Melakukan pengiriman uang
- Mengelola profil dan preferensi akun
- Mengakses fitur utama secara cepat dari dashboard

## User Flow
Flow utama:

```text
Splash
→ Onboarding
→ Sign In / Register
→ Home Dashboard
→ Statistics
→ My Cards
→ Transaction History
→ Search Transaction
→ Send Money
→ Profile / Settings
```

## Key Screens
### 1. Onboarding
Onboarding digunakan untuk memperkenalkan nilai utama aplikasi: cepat, aman, dan mudah digunakan.

### 2. Authentication
Sign in dan register dirancang dengan form sederhana agar pengguna dapat masuk atau membuat akun dengan cepat.

### 3. Home Dashboard
Dashboard menjadi pusat informasi utama yang menampilkan kartu, shortcut action, dan ringkasan transaksi.

### 4. Card Management
Screen kartu membantu pengguna melihat kartu yang dimiliki dan mengakses informasi terkait kartu.

### 5. Transaction History
Transaction history menampilkan daftar transaksi agar pengguna dapat memahami aktivitas keuangannya.

### 6. Search Transaction
Search transaction membantu pengguna menemukan transaksi tertentu dengan lebih cepat.

### 7. Send Money
Send money flow dirancang untuk membantu pengguna memilih penerima, memasukkan nominal, dan melihat detail pembayaran.

### 8. Profile & Settings
Profile dan settings mendukung pengelolaan akun, bahasa, password, dan informasi pengguna.

## Design System Summary
Arah visual project:

- Warna utama biru untuk CTA dan highlight
- Background putih untuk tampilan yang bersih
- Card-based layout untuk informasi finansial
- Bottom navigation untuk akses cepat
- Form input sederhana untuk auth dan transaksi
- Icon dan ilustrasi untuk memperjelas konteks
- Hierarki teks yang menonjolkan informasi penting

## Reflection
Project ini membantu menunjukkan kemampuan dalam menyusun flow mobile app, membuat tampilan financial dashboard, mendesain UI berbasis kartu, dan menyiapkan presentasi portfolio untuk recruiter.

Improvement yang disarankan sebelum dipublikasikan:

- Memperbaiki typo
- Menyamakan bahasa
- Menyamakan format mata uang
- Masking data dummy seperti nomor kartu, CVV, email, dan nomor HP
- Menggunakan video walkthrough yang lebih pendek
- Menampilkan screen terbaik di hero portfolio

## Disclaimer
UBank Digital adalah prototype UI/UX yang dibuat di Figma. Project ini tidak mencakup backend, database, integrasi banking API, payment processing, atau real banking operation.
