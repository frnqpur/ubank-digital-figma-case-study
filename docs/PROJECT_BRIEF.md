# UBank Digital — Project Brief

## Project Summary
UBank Digital adalah prototype UI/UX aplikasi mobile banking berbasis Figma. Project ini menampilkan pengalaman pengguna dari onboarding sampai aktivitas perbankan dasar seperti authentication, dashboard, card management, transaction history, send money, dan settings/profile.

Project ini paling tepat ditampilkan sebagai **UI/UX Mobile Banking Case Study**, bukan sebagai aplikasi mobile production atau sistem banking yang sudah terhubung ke backend.

## Project Type
- Category: UI/UX Design
- Platform: Mobile App
- Tool: Figma
- Deliverables:
  - Figma prototype
  - Mobile app screen design
  - Video demo / walkthrough
  - Portfolio case study
  - Screenshot assets

## Core Scope
Prototype mencakup flow utama berikut:

1. Splash / app intro
2. Onboarding
3. Sign in
4. Register
5. Dashboard / home
6. Statistics
7. My cards / card management
8. All cards
9. Transaction history
10. Search transaction
11. Send money
12. Profile
13. Edit profile
14. Settings
15. Language
16. Change password
17. Terms & conditions

## Main Focus
Fokus portfolio:
- UI/UX mobile banking
- Clean financial dashboard
- Simple onboarding
- Authentication screens
- Card-based banking interface
- Transaction list and search
- Send money flow
- Settings/profile management
- Figma prototype presentation
- Video demo for recruiter review

## Target User
Target user adalah pengguna mobile banking yang membutuhkan akses cepat untuk:
- Melihat informasi kartu dan saldo
- Mengecek transaksi
- Melakukan pengiriman uang
- Mengatur profil dan preferensi akun
- Memahami fitur utama aplikasi melalui onboarding

## Design Direction
Arah desain:
- Clean and modern mobile UI
- Primary blue color untuk CTA dan highlight
- White background untuk readability
- Card-based layout untuk financial information
- Bottom navigation untuk akses cepat
- Form input yang familiar untuk auth dan transaksi
- Visual onboarding dengan ilustrasi fintech

## Portfolio Positioning
Gunakan positioning berikut:

> UBank Digital is a Figma-based mobile banking UI/UX prototype designed to explore a simple and modern financial app experience, covering onboarding, authentication, dashboard, card management, transaction history, send money, and user settings.

## Important Disclaimer
Project ini adalah prototype desain UI/UX. Jangan mengklaim sebagai:
- Live banking app
